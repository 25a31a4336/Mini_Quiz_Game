# 🧠 Mini Quiz Game

A lightweight, console-based multiple-choice quiz written in **C**. Perfect for beginners learning C programming concepts while building a small, interactive project.

---

## ✨ Highlights

- Clean and simple interface — ideal for learning and demoing C fundamentals
- Instant answer validation and automatic scoring
- Easy to extend with more questions, categories, timers, and persistence

---

## 🎯 Features

- Multiple-choice questions
- Immediate feedback after each answer
- Automatic score calculation and final result display
- Minimal and user-friendly console UI

---

## 🛠️ Technologies Used

- **Language:** C
- **Suggested Compilers/IDEs:**
  - GCC
  - Code::Blocks
  - VS Code

---

## 📚 Concepts Demonstrated

This project helps beginners understand and practice:

- Loops (`for`, `while`)
- Conditional statements (`if/else`, `switch`)
- Arrays
- Functions and modular programming
- Basic input/output operations

---

## 🚀 Future Enhancements

Possible improvements for the project:

- Timer-based questions
- Randomized question order
- File-based score storage (Leaderboard)
- Multiple levels and categories
- Graphical or web-based frontend
